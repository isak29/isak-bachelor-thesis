# isak-bachelor-thesis

Repository för Isak Lampells kandidatuppsats om grafbaserade enterprise system, och hur de kan nyttjas av AI-agenter för att hitta och kontextualisera olika typer av data. Skapande av MCP server med tools. Ena settet tool används för att låta LLM'n passa in cypher frågor mot neo4j grafdatabasen, medans andra settet med tools exponeras mot LLM'n för att låta den kommunicera och hämta efterfrågad information från olika externa applikationer via deras API:er.
