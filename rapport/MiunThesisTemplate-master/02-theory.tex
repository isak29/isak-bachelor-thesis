\subsection{Kunskapsgraf}\label{subsec:Kunskapsgraf}
En kunskapsgraf är ett designmönster som som används för att organisera och representera data där fokus ligger på relationer mellan olika entiteter.
Istället för att lagra information i tabeller eller dokument, struktureras data i form av noder (entiteter) och relationer som kopplar samman dessa.
Detta gör det möjligt att analysera och förstå komplexa samband i data på ett effektivt vis\cite{5}.

Kunskapsgrafer är särskilt användbara i situationer där data är starkt sammankopplad och där relationerna i sig bär viktig information. De används inom rekommendationssystem, bedrägeridetektion och avancerad informationssökning.
Genom att integrera data från olika källor och organisera den efter tydliga samband kan kunskapsgrafer bidra till att synliggöra mönster och insikter som annars kan vara svåra att upptäcka\cite{5}.


\subsection{MCP}\label{subsec:mcp_theory}
MCP (Model Context Protocol) är ett protokoll som används för att låta applikationer förse LLM'er med kontextuell information.
MCP gör det möjligt för olika AI-modeller att ansluta till olika datakällor och verktyg. Detta gör det enklare att utveckla AI-applikationer utan att behöva skapa specialiserade gränssnitt för varje datakälla eller modell\cite{4}.

MCP består av tre huvudroller: servrar, klienter och värdar. Servrarna är programmen som tillhandahåller verktyg och dataåtkomstmöjligheterna för LLM'er att använda.
Varje MCP server tillhandahåller en uppsättning verktyg, vilka är de som ansvarar för att hämta informationen från datakällorna. Klienterna är de som sammankopplar LLM'en med MCP servern. De ansvarar för att ta emot förfrågningar från LLM'en, vidarebefordra dessa till motsvarande MCP server och returnera resultatet från servern tillbaka till LLM'en.
Värdarna är de som vill få åtkomst till data genom MCP. Exempelvis program som tillhandahåller ett användargränssnitt för att interagera med LLM'er, medan de själva integrerar MCP klienten för att kunna ansluta MCP servrarna och därmed utöka LLM'ens funktion med de verktyg som tillhandahålls av servrarna\cite{4}.
